## What Is MyCryptoCheckout?

MyCryptoCheckout is a WordPress plugin that plugs into WooCommerce and Easy Digital Downloads, turning your store into a direct crypto payment receiver. When a customer pays, the coins go straight to your wallet — MCC's servers never touch the money. They just handle the blockchain confirmation side of things via their API.

It supports over 100 cryptocurrencies: Bitcoin, Ethereum, BNB, Solana, Litecoin, Dash, DigiByte, and a huge list of ERC20 tokens. If you want to accept some more obscure coin, there's even a form to request additions.

The plugin has been around long enough to process over $777 million in transaction value across thousands of stores. That's not a tiny side project.

## The Key Features That Actually Matter

**Zero percent transaction fees.** Not "low fees." Not "competitive fees." Zero. You don't pay per transaction, ever. The only cost is the optional annual or lifetime license.

**Peer-to-peer, no redirect.** Customers never leave your site to complete payment. No iframe, no redirect to a third-party page. The checkout happens right there.

**No KYC required.** You don't need to submit ID documents, business verification, or anything like that. Install the plugin, enter your wallet address, done.

**HD wallet support.** Hierarchically deterministic wallets mean MCC can detect unique payments even if multiple customers send to the same address — a smart solution to what would otherwise be a real headache.

**1-click wallet buttons.** MetaMask, Trust Wallet, Electrum, Phantom — customers can pay with one click from their browser wallet.

**Fiat auto-settlement.** If you're worried about crypto price volatility, this feature can automatically convert incoming crypto to fiat at settlement. Useful if you want the benefits of crypto acceptance without holding volatile assets.

**Donations widget.** There's a shortcode generator for a donations widget you can embed on any page. Great for blogs, nonprofits, or content creators.

**Security suite.** Recent updates added admin lockdown, disabled file editors, XML-RPC blocking, and wallet change watchdog alerts. These are actually thoughtful security additions, not just checkbox features.

## Setup: How Fast Can You Actually Get Running?

Multiple users report being live in under 5 minutes. The process is basically:

1. Install the plugin from WordPress.org
2. Activate it — this creates your account on the MCC API server automatically
3. Go to Settings → MyCryptoCheckout and enter your wallet addresses for whichever coins you want to accept
4. Enable it as a payment method in WooCommerce

That's it. No API keys to hunt down, no merchant account approval, no waiting period.
